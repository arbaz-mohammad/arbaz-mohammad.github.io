My Projects

Automatic Plant monitor(Click here to view)

An automatic plant monitor using NodeMCU is a system that utilizes the NodeMCU microcontroller to collect data on the moisture levels, temperature, and humidity of a plant's environment. The collected data is transmitted to a cloud-based platform where it can be analyzed and used to make decisions about the plant's care. This system allows for real-time monitoring and can alert users when the plant needs watering or other care.
